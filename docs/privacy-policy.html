Co-Talk 개인정보 처리방침

1. 개인정보의 처리 목적

Co-Talk(이하 "회사")은 다음의 목적을 위하여 개인정보를 처리합니다:

  • 회원 가입 및 관리: 회원제 서비스 이용에 따른 본인확인, 개인식별, 불량회원의 부정이용 방지, 회원 탈퇴 시 본인 확인
  • 서비스 제공: 실시간 메시징 서비스, 채팅방 생성 및 관리, 친구 관리, 프로필 관리, 메시지 전송 및 수신, 읽음 확인, 이모지 반응, 미디어 파일 공유, 링크 미리보기, 온라인 상태 표시, 타이핑 상태 표시
  • 알림 서비스 제공: 푸시 알림을 통한 메시지 수신 알림, 친구 요청 알림, 채팅방 초대 알림
  • 서비스 개선 및 개발: 서비스 이용 통계 분석, 서비스 품질 개선
  • 고객 지원: 민원 처리, 고지사항 전달, 분쟁 조정

2. 수집하는 개인정보 항목

회사는 서비스 제공을 위해 다음의 개인정보를 수집합니다:

가. 회원가입 시 수집하는 필수 항목

  • 이메일 주소 (계정 식별 및 로그인 용도)
  • 비밀번호 (암호화하여 서버에 저장, 단말기에는 저장하지 않음)
  • 닉네임 (서비스 내 사용자 표시명)

나. 선택적으로 수집하는 항목

  • 프로필 사진 (사용자가 직접 선택 또는 촬영하여 업로드)
  • 상태 메시지 (사용자가 직접 작성)
  • 배경 이미지 (프로필 꾸미기 용도)

다. 서비스 이용 과정에서 생성 또는 수집되는 항목

  • 채팅 메시지 내용 (텍스트, 이미지, 파일 등)
  • 메시지 메타데이터 (전송 시간, 수신 시간, 읽음 확인 여부, 이모지 반응)
  • 채팅방 정보 (참여자 목록, 채팅방 생성 일시, 채팅방 이름)
  • 친구 목록, 친구 요청 내역, 차단 사용자 목록
  • 사용자 검색 기록
  • 알림 설정 (알림 유형별 on/off, 방해금지 시간 설정)
  • 미디어 파일 (사진, 동영상, 문서 등 채팅 중 공유한 첨부파일)
  • 링크 미리보기 데이터 (메시지에 포함된 URL의 제목, 설명, 이미지)

라. 자동으로 수집되는 항목

  • 기기 유형 (Android 또는 iOS)
  • FCM 푸시 알림 토큰 (디바이스 식별 및 푸시 알림 발송 용도)
  • 접속 일시, 온라인 상태, 마지막 접속 시간
  • 타이핑 상태 정보 (실시간 타이핑 알림 표시 용도)

3. 개인정보의 수집 방법

회사는 다음의 방법으로 개인정보를 수집합니다:

  • 회원가입 시 사용자가 직접 입력
  • 서비스 이용 과정에서 사용자의 선택 또는 작성에 의해 수집
  • 애플리케이션 이용 시 자동 생성 및 수집 (접속 기록, 온라인 상태, FCM 토큰 등)

4. 단말기 접근 권한

Co-Talk은 다음의 단말기 접근 권한을 요청합니다. 모든 권한은 서비스 이용을 위해 필요한 경우에만 요청하며, 거부하시더라도 해당 기능을 제외한 서비스는 이용 가능합니다.

  • 카메라 (선택): 프로필 사진 촬영, 채팅 중 사진 촬영 기능 제공
  • 사진 라이브러리 (선택): 프로필 사진 선택, 채팅 중 이미지 선택 및 저장 기능 제공
  • 알림 (선택): 푸시 알림 수신을 통한 메시지 및 친구 요청 알림 제공
  • 생체인증 - Face ID/Touch ID (선택): 앱 잠금 기능 사용 시 생체 인증을 통한 앱 보안 강화

위 권한은 Android 및 iOS 운영체제 설정에서 언제든지 변경하실 수 있습니다.

5. 개인정보의 보유 및 이용기간

회사는 회원 탈퇴 시 또는 개인정보 수집 및 이용목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다.

가. 회원정보

  • 인증 토큰, 사용자 ID, 이메일: 로그인 유지를 위해 기기의 보안 저장소(iOS Keychain / Android Keystore)에 보관하며, 로그아웃 시 즉시 삭제
  • 프로필 정보(닉네임, 프로필 사진, 상태 메시지 등): 회원 탈퇴 시까지 서버에 보관
  • 비밀번호: 서버 전송 시 암호화되며, 단말기에는 저장되지 않음

나. 채팅 데이터

  • 채팅 메시지, 미디어 파일, 채팅방 정보: 서버에 보관되며, 사용자가 채팅방을 나가거나 회원 탈퇴 시 삭제

다. 법령에 따른 보존

다만, 관계법령의 규정에 의하여 보존할 필요가 있는 경우 회사는 관계법령에서 정한 일정한 기간 동안 회원정보를 보관합니다:

  • 계약 또는 청약철회 등에 관한 기록: 5년 (전자상거래법)
  • 소비자의 불만 또는 분쟁처리에 관한 기록: 3년 (전자상거래법)
  • 접속에 관한 기록: 3개월 (통신비밀보호법)

6. 개인정보의 파기

회사는 개인정보 보유기간의 경과, 처리목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체 없이 해당 개인정보를 파기합니다.

가. 파기 절차

  • 회원 탈퇴 시 비밀번호 확인 후 모든 개인정보 삭제
  • 법령에 따라 보존이 필요한 정보는 별도 저장 후 분리 보관

나. 파기 방법

  • 전자적 파일 형태의 정보는 기록을 재생할 수 없는 기술적 방법을 사용하여 영구 삭제
  • 종이에 출력된 개인정보는 분쇄기로 분쇄하거나 소각하여 파기

7. 개인정보의 제3자 제공

회사는 원칙적으로 정보주체의 개인정보를 제3자에게 제공하지 않습니다.

가. Firebase Cloud Messaging (Google) 제공

회사는 푸시 알림 서비스 제공을 위해 다음과 같이 개인정보를 제3자에게 제공합니다:

  • 제공받는 자: Firebase Cloud Messaging (Google Inc.)
  • 제공 목적: 푸시 알림 전송 (메시지 수신, 친구 요청, 채팅방 초대 알림)
  • 제공 항목: FCM 푸시 알림 토큰, 기기 유형 (Android/iOS)
  • 보유 및 이용기간: 회원 탈퇴 또는 푸시 알림 서비스 해지 시까지
  • 거부 권리: 정보주체는 푸시 알림 설정을 통해 언제든지 제공을 거부할 수 있으며, 거부 시 푸시 알림을 제외한 서비스는 정상적으로 이용 가능합니다.

※ 회사는 Firebase Analytics 또는 기타 사용자 행동 추적 서비스를 사용하지 않으며, 푸시 알림 전송 목적으로만 Firebase를 사용합니다.

나. 법령에 의한 제공

다음의 경우에는 예외적으로 개인정보를 제공할 수 있습니다:

  • 정보주체가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

8. 개인정보의 안전성 확보 조치

회사는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 관리적 조치: 개인정보 취급 직원의 최소화 및 교육, 내부관리계획 수립 및 시행
  • 기술적 조치:
    • HTTPS/WSS 프로토콜을 통한 암호화 통신
    • JWT(JSON Web Token) 기반 인증 시스템
    • iOS Keychain 및 Android Keystore를 활용한 인증 토큰 안전 보관
    • 비밀번호 서버 전송 시 암호화, 단말기에 비밀번호 미저장
    • 개인정보 처리 시스템 접근권한 관리
    • 접속기록의 위조·변조 방지를 위한 보안기능 사용
  • 물리적 조치: 개인정보 보관 시설의 접근통제

9. 정보주체의 권리

정보주체는 회사에 대해 언제든지 다음의 개인정보 보호 관련 권리를 행사할 수 있습니다:

  • 개인정보 열람 요구 (프로필 페이지에서 확인 가능)
  • 오류 등이 있을 경우 정정 요구 (프로필 편집 기능)
  • 삭제 요구 (회원 탈퇴 기능, 비밀번호 확인 필요)
  • 처리정지 요구 (알림 설정 비활성화)

위 권리 행사는 서비스 내 설정 메뉴를 통해 직접 하실 수 있으며, 고객센터(privacy@cotalk.com)를 통해서도 요청하실 수 있습니다.

정보주체가 개인정보의 오류 등에 대한 정정 또는 삭제를 요구한 경우에는 회사는 정정 또는 삭제를 완료할 때까지 당해 개인정보를 이용하거나 제공하지 않습니다.
